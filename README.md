## Run Locally

Clone the project

```bash
  git clone https://github.com/Mudassir710/Manage-landing-page.git
```

Go to the project directory

```bash
  cd Manage-landing-page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
